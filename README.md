# Fashion Store — متجر هدوم

متجر إلكتروني لمحل ملابس (زي Shein) — العميل يختار، والطلب يتبعت لواتساب صاحب المحل.

## 📍 مكان المشروع على اللاب

```
D:\fashion-store
```

## 🚀 تشغيل الموقع

```powershell
cd D:\fashion-store
npm install
npm run dev
```

افتحي الرابط اللي هيظهر (غالباً `http://localhost:5173`).

## ⚙️ أهم إعداد (لازم تعمليه)

افتحي الملف `src/config/store.js` وغيّري:

```js
export const STORE_WHATSAPP = '201012345678'  // رقم واتساب المحل
```

- كود مصر: `20` + الرقم من غير صفر في الأول
- مثال: لو الرقم `01012345678` → `201012345678`

## 🛒 إزاي الشغل بيمشي

1. **العميل** يدخل الموقع → يتصفّح المنتجات → يضيف للسلة
2. **Checkout** → يكتب اسمه، موبايله، عنوانه
3. **زر واتساب** → يفتح واتساب برسالة فيها كل تفاصيل الطلب
4. **صاحب المحل** يستلم الرسالة → يحضّر الطلب → يبعته مع المندوب
5. **العميل** يدفع **كاش عند الاستلام**

## 📁 هيكل المشروع

| ملف/مجلد | الوظيفة |
|-----------|---------|
| `src/data/products.js` | المنتجات (ضيفي/عدّلي هنا) |
| `src/config/store.js` | رقم واتساب، رسوم التوصيل، المدن |
| `src/i18n/locales/` | ترجمة عربي / إنجليزي |
| `src/components/` | واجهة الموقع |
| `src/utils/whatsapp.js` | تجهيز رسالة الطلب |

## ➕ إضافة منتج جديد

في `src/data/products.js` انسخي object منتج وعدّلي:

```js
{
  id: 9,
  nameAr: 'اسم بالعربي',
  nameEn: 'Name in English',
  category: 'women', // women | men | kids
  price: 300,
  sizes: ['S', 'M', 'L'],
  colors: ['أسود'],
  colorsEn: ['Black'],
  image: 'https://...',
  inStock: true,
}
```

## 🌐 نشر الموقع على الإنترنت (لاحقاً)

| الخطوة | إزاي |
|--------|------|
| بناء نسخة production | `npm run build` → مجلد `dist` |
| استضافة مجانية | [Netlify](https://netlify.com) أو [Vercel](https://vercel.com) — ارفعي مجلد `dist` |
| دومين خاص | اشتري دومين واربطيه بالاستضافة |

## ❗ حاجات ناقصة (وإزاي تتعمل)

| الحاجة | الوضع الحالي | إزاي تتعمل |
|--------|-------------|------------|
| **صور منتجاتك الحقيقية** | صور تجريبية من Unsplash | حطي روابط صورك في `products.js` أو مجلد `public/images/` |
| **رقم واتساب المحل** | رقم تجريبي | غيّريه في `store.js` |
| **لوحة تحكم لصاحب المحل** | مش موجودة (اخترتي واتساب بس) | ممكن نضيف صفحة admin لاحقاً |
| **تتبع الطلب للعميل** | مش موجود | يحتاج backend + قاعدة بيانات |
| **دفع أونلاين** | مش موجود (كاش بس) | Paymob / Fawry API |
| **مندوب التوصيل** | يدوي | صاحب المحل ينسّق من واتساب |
| **Node.js** | لازم يكون مثبت | [nodejs.org](https://nodejs.org) |
